基本设置
*联盟名称：
显示在整站会出现联盟名称的地方
*联盟域名：
联盟域名的根域，比如 www.zyiis.com
*联盟广告标识：
联盟广告标识,附加在广告上的一个联盟标志
提现设置
*每月允许提现次数：
限制网站主提现次数。请填写数字，0为不限制
*允许提现的周天：
限制网站主提现周天。1-7代表星期一至星期天，英文逗号分隔，0为不限制
*允许提现的时间：
限制网站主提现时间段。24小时制，0为不限制，例如：9-18
*最低提现额：
限制网站主提现最低金额。请填写数字，例如：20
*个税税率： %
个人所得税税率。以百分比计算，请填写数字，例如：10
*提现手续费计费方式： 固定额 百分比
*提现手续费：
财务结算根据实付金额(税后)扣除相应手续费

*开放网站主注册： 开启 关闭

*开放广告主注册： 开启 关闭

*注册验证方式： 自动通过 邮件验证 人工验证
邮件验证将向用户注册 Email 发送一封验证邮件以确认邮箱的有效性
*前台登录验证码： 开启 关闭
防恶意频繁登录

积分设置
*积分开关： 开启 关闭
*积分商城开关： 开启 关闭
*积分方式： 提现积分 PV积分 提现和PV均有积分
