小贴士

  • 1.佣金满多少元可以提现？

    佣金只需满20元（含20元）即可申请提现。如果您当月佣金未达到20元，则自动累计到下周。

  • 2.佣金每月允许提现次数？

    佣金每月允许提现次数:4次

  • 3.佣金允许提现周天？

    佣金允许提现周天:星期一,星期二,星期三,星期四,星期五

  • 3.佣金允许提现时间？

    佣金允许提现时间:从9点到18点
